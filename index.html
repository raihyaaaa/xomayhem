How to Play

Getting Started

  • Choose Symbol: Select X or O
  • Select Grid Size: Pick 3×3, 5×5, or 7×7
  • Select Difficulty: Easy, Medium, or Hard
  • Click Start: Begin your game

Basic Rules

  • Players take turns placing symbols on the grid
  • X always goes first
  • Click any empty square to place your symbol
  • Get the required symbols in a row to win:
    • 3×3 grid: 3 in a row
    • 5×5 grid: 4 in a row
    • 7×7 grid: 5 in a row
  • Rows can be horizontal, vertical, or diagonal
  • New game starts automatically after 2 seconds

Winning Strategies

  • Control Center: Center position is strategic
  • Block Opponent: Watch for opponent's patterns
  • Create Threats: Set up multiple winning moves
  • Think Ahead: Plan 2-3 moves in advance

Difficulty Levels

Easy: Random moves - great for beginners

Medium: Mix of strategy and random - balanced challenge

Hard: Optimal strategy - very difficult!
